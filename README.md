# dating-app
Demonstrates CRUD operation (including photo upload) with Angular (frontend) and ASP.NET Core (backend).

## Development Environment
- npm: 6.7.0
- Node: 11.10.1
- Angular CLI: 6.0.8
  - npm install
  - ng build
  - ng serve
  - localhost:4200

- DotNet Core SDK: 2.1.3
  - dotnet build
  - dotnet ef database update
  - dotnet watch run
  - localhost:5000
